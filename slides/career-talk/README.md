# 進路講演スライド

高校生向けITエンジニア進路講演（80分）

## ファイル構成

```
career-talk/
├── _config.md       # 共通設定（frontmatter）
├── 00-opening.md    # オープニング（3分）
├── 01-self-intro.md # 1章：自分の話（12〜15分）
├── 02-it-jobs.md    # 2章：ITの仕事（15〜18分）
├── 03-line-tech.md  # 3章：LINE技術深掘り（20〜25分）
├── 04-ai-era.md     # 4章：AI時代（12〜15分）
└── 05-closing.md    # まとめ・Q&A（11〜13分）
```

## コマンド

```bash
# プレビュー（結合してブラウザで表示）
pnpm preview:career-talk

# HTML出力
pnpm build:career-talk

# PDF出力
pnpm pdf:career-talk

# PowerPoint出力
pnpm pptx:career-talk
```

## 編集方法

各セクションのmdファイルを直接編集してください。
`<!-- 【要編集】 -->` コメントがある箇所は、あなたの情報に書き換えてください。

### 編集が必要な箇所

| ファイル | 箇所 |
|---------|------|
| 00-opening.md | 発表者名・所属 |
| 01-self-intro.md | プロフィール、高校時代、大学、初めて作ったもの |
| 05-closing.md | 最後のメッセージ、Q&A回答、連絡先 |

## 時間配分（80分想定）

| セクション | 時間 |
|-----------|------|
| 0. オープニング | 3分 |
| 1. 自分の話 | 12〜15分 |
| 2. ITの仕事 | 15〜18分 |
| 3. LINE技術 | 20〜25分 |
| 4. AI時代 | 12〜15分 |
| 5. まとめ・Q&A | 11〜13分 |

### 70分版への調整

- 2章のワークを5分に短縮
- 3章のワークを7分に短縮
- Q&Aを5分に短縮
