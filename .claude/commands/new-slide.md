# 新しいスライドを作成

ユーザーが指定したトピックに基づいて、新しいスライドを作成します。

## 手順

1. ユーザーにスライドのトピックと目的を確認
2. `templates/basic.md` をベースにして新しいスライドを作成
3. `slides/` ディレクトリに保存
4. `theme.css` のコンポーネントを適切に活用

## ルール

- タイトル: 最大30文字
- 1スライドあたり: 最大400文字
- 1スライドに1メッセージ
- 視覚的なコンポーネントを積極的に使用

## 出力

`slides/{topic}.md` にスライドを作成してください。
