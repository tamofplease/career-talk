# スライドをビルド

Marp CLIを使用してスライドを各種フォーマットに出力します。

## 出力形式

- **HTML**: `pnpm build`
- **PDF**: `pnpm pdf`
- **PowerPoint**: `pnpm pptx`

## 手順

1. 出力形式を確認
2. 対応するpnpmスクリプトを実行
3. `dist/` ディレクトリに出力を確認

## コマンド

```bash
# HTML出力
pnpm build

# PDF出力
pnpm pdf

# PowerPoint出力
pnpm pptx
```

出力ファイルは `dist/` ディレクトリに保存されます。
