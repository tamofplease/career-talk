# スライドのアウトラインを作成

指定されたトピックについて、スライドのアウトライン（構成案）を作成します。

## 手順

1. トピックの目的と対象者を確認
2. 適切なスライド構成を提案
3. 各スライドの概要を記述
4. 使用推奨コンポーネントを提案

## アウトラインのフォーマット

```
## スライド構成案

### 1. タイトル (title)
- メインタイトル:
- サブタイトル:

### 2. アジェンダ (agenda)
- 項目1
- 項目2
- 項目3

### 3. 本編1
- コンポーネント: [推奨コンポーネント]
- 内容:

...

### N. まとめ (conclusion)
- キーポイント
```

## 考慮事項

- 発表時間に応じたスライド枚数
- 対象者のレベルに合わせた内容
- 視覚的な要素のバランス
