# スライドを改善

既存のスライドを読み込み、より効果的なプレゼンテーションに改善します。

## 手順

1. 指定されたスライドファイルを読み込む
2. 以下の観点で分析:
   - 文字数（タイトル30文字、本文400文字以内か）
   - 1スライド1メッセージになっているか
   - 視覚的なコンポーネントの活用
   - 情報の流れと論理構成
3. 改善案を提案
4. ユーザーの承認後に修正を適用

## 改善チェックリスト

- [ ] タイトルは簡潔か（30文字以内）
- [ ] 各スライドは1メッセージに絞られているか
- [ ] 文字数は適切か（400文字以内）
- [ ] 視覚的なコンポーネントを活用しているか
- [ ] 情報の階層が明確か
- [ ] 余白は十分か

## 使用可能なコンポーネント

- レイアウト: two-column, split-image, strategy-grid
- 数値表示: metric-card, stat-highlight
- 比較: comparison-matrix, pros-cons, before-after
- プロセス: timeline, steps, flow-chart
- 強調: highlight-box, quote-box, alert
